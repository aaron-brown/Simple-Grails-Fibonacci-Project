Clone using:

git clone <link>
In springsource, import the project
Click run as -> run-app

On the index page, type in a number and hit submit
The fibonacci sequence will be displayed
  Please note that after 2^(32-1) has been reached, you will get sign-flips
If you submit a blank form, a sequence of 10 will be assumed as the default.

There may be merge errors or conflicts in the .class files. These are ignorable
Since those get rebuilt on the next run. Derp.
